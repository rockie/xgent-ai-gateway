       - `pub async fn ensure_consumer_group(&self, service: &ServiceName) -> Result<(), GatewayError>` -- calls `xgroup_create_mkstream` on `tasks:{service}` with group "workers" and start ID "0", ignores BUSYGROUP error per Pitfall 1
       - `pub async fn submit_task(&self, service: &ServiceName, payload: Vec<u8>, metadata: HashMap<String, String>) -> Result<TaskId, GatewayError>`:
         - Generate TaskId::new()
         - Ensure consumer group exists for this service (per D-02, lazy creation per Open Question 3)
         - Redis pipeline: HSET `task:{task_id}` with fields: state="pending", service=service, payload=base64(payload), metadata=serde_json(metadata), created_at=chrono::Utc::now().to_rfc3339()
         - XADD to `tasks:{service}` with MAXLEN ~10000 (approximate trimming) entry `task_id={task_id}`
         - EXPIRE `task:{task_id}` with result_ttl_secs
         - Return task_id
       - `pub async fn get_task_status(&self, task_id: &TaskId) -> Result<TaskStatus, GatewayError>`:
         - HGETALL `task:{task_id}`, return TaskNotFound if empty
         - Parse fields into `TaskStatus { task_id, state: TaskState, payload, result, error_message, metadata, created_at, completed_at }`
       - `pub async fn report_result(&self, task_id: &TaskId, success: bool, result: Vec<u8>, error_message: String) -> Result<(), GatewayError>`:
         - Verify task exists and state is Assigned or Running
         - HSET `task:{task_id}`: state=completed/failed, result=base64(result), error_message, completed_at=now
         - XACK the stream entry (retrieve service and stream_id from hash)
       - `pub async fn poll_task(&self, service: &ServiceName, node_id: &str) -> Result<Option<TaskAssignmentData>, GatewayError>`:
         - XREADGROUP with group="workers", consumer=node_id, COUNT 1, BLOCK block_timeout_ms on `tasks:{service}`
         - If task received: parse task_id from entry, HSET `task:{task_id}` state="assigned", return TaskAssignmentData { task_id, payload, metadata }
         - If timeout: return None
       - Define `pub struct TaskStatus` and `pub struct TaskAssignmentData` with appropriate fields
