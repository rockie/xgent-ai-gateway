    ```rust
    pub fn extract_json_value(root: &serde_json::Value, path: &str) -> Result<String, String> {
        let segments: Vec<&str> = path.split('.').collect();
        let mut current = root;
        for segment in &segments {
            if let Ok(index) = segment.parse::<usize>() {
                current = current.get(index).ok_or_else(|| {
                    format!("array index {} out of bounds at path '{}'", index, path)
                })?;
            } else {
                current = current.get(*segment).ok_or_else(|| {
                    format!("key '{}' not found at path '{}'; response: {}",
                        segment, path, serde_json::to_string(root).unwrap_or_default())
                })?;
            }
        }
        match current {
            serde_json::Value::String(s) => Ok(s.clone()),
            serde_json::Value::Null => Ok("null".to_string()),
            other => Ok(serde_json::to_string(other).unwrap_or_default()),
        }
    }
    ```

    **5. Executor::execute() implementation (per D-01 through D-14):**
    Flow:
    a. `let mut variables = placeholder::build_task_variables(assignment, &self.service_name);`
    b. Resolve URL: `placeholder::resolve_placeholders(&self.sync_api.url, &variables)?` -- supports `<service_name>`, `<payload>`, etc. in URL (per open question 1 from RESEARCH.md -- YES support task placeholders)
    c. Resolve body template if present: `placeholder::resolve_placeholders(body_template, &variables)?`
    d. Resolve header values: iterate `self.sync_api.headers`, resolve each value via `placeholder::resolve_placeholders`, build `reqwest::header::HeaderMap`
    e. Parse method: `reqwest::Method::from_bytes(self.sync_api.method.to_uppercase().as_bytes())` with fallback to POST (per Pitfall 4)
    f. Send HTTP request with one-retry on connection failure:
       - First attempt: build and send request
       - If `Err(e)` where `e.is_connect()` (NOT is_timeout per Pitfall 2): log warning, retry once
       - If retry also fails: return failure with "HTTP request failed after retry: {e}"
       - If `e.is_timeout()`: return failure with "HTTP request timed out after {N}s" (per D-14, D-05)
       - Other errors: return failure with "HTTP request failed: {e}"
    g. Read response body as text ONCE (per Pitfall 1): `let body_text = response.text().await?`
    h. Check HTTP status (per D-12): if `!status.is_success()`, return failure with `"HTTP {status_code}: {body_text}"`
    i. Check body_text length against max_bytes (per Pitfall 3 and open question 2): if `body_text.len() > self.response.max_bytes`, return failure
    j. Parse body as JSON: `serde_json::from_str::<serde_json::Value>(&body_text)`
    k. Scan response body template for `<response.XXX>` placeholders, extract each from JSON via `extract_json_value`, insert "response.XXX" -> value into variables map (per D-11)
    l. Call `response::resolve_response_body(&self.response.body, &variables, self.response.max_bytes)` for final assembly
    m. Return `ExecutionResult { success: true, result: bytes, error_message: String::new() }`
