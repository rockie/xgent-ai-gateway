   - Remove `base64::Engine::decode(...)` for payload -- use the raw string directly: `let payload = fields.get("payload").cloned().unwrap_or_default();`

5. In `report_result` method:
   - Change parameter: `result: Vec<u8>` -> `result: String`
   - Remove `base64::Engine::encode(...)` for result
   - Store `result` directly: `.arg("result").arg(&result)`

6. Remove `use base64::Engine as _;` import if present (check if base64 is used elsewhere in the file -- it should not be after these changes).

7. Update unit tests in the same file:
   - `submit_task` test: change `b"hello world".to_vec()` to `"\"hello world\"".to_string()` (JSON-encoded string)
   - Update any `assert_eq!(status.payload, ...)` to compare String values
   - Any `report_result` call: change `b"...".to_vec()` to `"...".to_string()`
