**1. Modify gateway/src/grpc/poll.rs** to validate node tokens (AUTH-03):

In `poll_tasks` method, BEFORE the existing service name validation and streaming setup:
- Extract token from gRPC metadata: `request.metadata().get("authorization")` -> parse `Bearer <token>` to get raw token
- If no authorization metadata or not Bearer format: return `Err(Status::unauthenticated("missing or invalid auth token"))` with `tracing::debug!("Node poll rejected: missing auth token")`
- Call `crate::auth::node_token::validate_node_token(&mut self.state.auth_conn.clone(), &req.service_name, raw_token).await`
- If validation returns `Ok(false)`: return `Err(Status::unauthenticated("unauthorized"))` with `tracing::debug!("Node poll rejected: invalid token for service {}", req.service_name)`
- If validation returns `Err(e)`: return `Err(Status::internal("internal error"))` with `tracing::error!("Redis error during node auth: {}", e)`
- Extract token BEFORE calling `request.into_inner()` -- get metadata reference first, then into_inner

The key change: get the token from metadata before consuming the request:
```rust
async fn poll_tasks(&self, request: Request<PollTasksRequest>) -> Result<Response<Self::PollTasksStream>, Status> {
    // Extract auth token from metadata BEFORE into_inner
    let raw_token = request.metadata()
        .get("authorization")
        .and_then(|v| v.to_str().ok())
        .and_then(|v| v.strip_prefix("Bearer "))
        .map(|s| s.to_string())
        .ok_or_else(|| {
            tracing::debug!("Node poll rejected: missing auth token");
            Status::unauthenticated("unauthorized")
        })?;

    let req = request.into_inner();
    // ... existing service name validation ...

    // Validate token against Redis
    let token_valid = crate::auth::node_token::validate_node_token(
        &mut self.state.auth_conn.clone(),
        &req.service_name,
        &raw_token,
    ).await.map_err(|e| {
        tracing::error!("Redis error during node auth: {}", e);
        Status::internal("internal error")
    })?;

    if !token_valid {
        tracing::debug!(service=%req.service_name, "Node poll rejected: invalid token");
        return Err(Status::unauthenticated("unauthorized"));
    }

    // ... rest of existing poll_tasks logic unchanged ...
}
```
